## Опис проєкту

DormManager — це веб-застосунок (MVP) для автоматизації роботи студентського гуртожитку. Система дозволяє коменданту та адміністрації зручно керувати житловим фондом, а студентам — швидко вирішувати побутові питання.

### Основні функції MVP:

- **Оплата:** автоматичне зняття коштів за проживання.
- **Облік кімнат:** перегляд вільних/зайнятих місць.
- **Поселення/Виселення:** керування профілями студентів.
- **Заявки на ремонт:** студенти можуть залишати тікети для електрика чи сантехніка.

## Інструкція із запуску (Локально)

1. Склонуйте репозиторій:
   `git clone https://github.com/ВАШ_ЮЗЕРНЕЙМ/назва-репо.git`
2. Перейдіть у папку проєкту:
   `cd назва-репо`
3. Відкрийте файл `index.html` у вашому браузері.
